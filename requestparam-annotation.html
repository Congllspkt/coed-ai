@RequestParam Annotation in Spring Boot

What is @RequestParam?

The @RequestParam annotation in Spring Boot is used to extract query parameters, form data, or any custom parameters from the URL of an HTTP request and bind them to method parameters in a controller.

Example Usage

@RestController
public class UserController {

    @GetMapping("/users")
    public ResponseEntity<String> getUserById(@RequestParam String id) {
        // Process the id parameter
        return ResponseEntity.ok("User ID: " + id);
    }
}

In the above example, the @RequestParam annotation is used to bind the id query parameter from the URL to the method parameter.

How it Works

When a request is made to the endpoint, Spring automatically maps the query parameter in the URL to the method parameter annotated with @RequestParam. If the parameter is not provided, a default value can be specified or an exception will be thrown if the parameter is required.
