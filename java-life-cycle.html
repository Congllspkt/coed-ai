Java Life Cycle and Its Components

The Java life cycle involves several stages that a Java program goes through from writing the code to execution. These stages include:

  • Writing the Code: The programmer writes the Java code in a text editor or an Integrated Development Environment (IDE).
  • Compilation: The Java compiler (javac) converts the source code into bytecode.
  • Class Loading: The Java ClassLoader loads the bytecode into the JVM.
  • Bytecode Verification: The JVM verifies the bytecode for security and correctness.
  • Execution: The JVM interprets or compiles the bytecode into machine code and executes it.

Examples

Compilation and Execution


// Save this file as HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// Compilation
// Run the following command in the terminal:
// javac HelloWorld.java

// Execution
// Run the following command in the terminal:
// java HelloWorld

Class Loading and Execution


public class ClassLoadingExample {
    static {
        System.out.println("Class is being loaded");
    }

    public static void main(String[] args) {
        System.out.println("Class is being executed");
    }
}

// Output:
// Class is being loaded
// Class is being executed
