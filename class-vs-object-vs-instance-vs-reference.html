Class vs Object vs Instance vs Reference in Java

Class

A class is a blueprint or template for creating objects. It defines properties and behaviors that the objects created from the class will have.


class Car {
    String brand;
    int speed;

    void drive() {
        System.out.println("Driving " + brand);
    }
}

Object

An object is an instance of a class. It is a real-world entity that has state and behavior.


Car myCar = new Car();
myCar.brand = "Toyota";
myCar.speed = 120;
myCar.drive();

Instance

An instance refers to the actual object created from a class. It occupies memory and has a unique identity.


Car anotherCar = new Car(); // another instance of the Car class
anotherCar.brand = "Honda";
anotherCar.speed = 100;
anotherCar.drive();

Reference

A reference is a variable that holds the memory address of an object. It is used to access the object.


Car carReference = myCar; // carReference points to the same object as myCar
carReference.drive();
