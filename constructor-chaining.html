Constructor Chaining in Java

Constructor chaining in Java is the process of calling one constructor from another constructor within the same class or from a parent class. It helps in reusing the code and reducing redundancy.

Examples

Example 1: Constructor Chaining within the Same Class


public class Employee {
    private String name;
    private int age;

    // Constructor with no arguments
    public Employee() {
        this("Unknown", 0); // Calls the constructor with two arguments
    }

    // Constructor with one argument
    public Employee(String name) {
        this(name, 0); // Calls the constructor with two arguments
    }

    // Constructor with two arguments
    public Employee(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void displayInfo() {
        System.out.println("Name: " + name + ", Age: " + age);
    }

    public static void main(String[] args) {
        Employee emp1 = new Employee();
        Employee emp2 = new Employee("Alice");
        Employee emp3 = new Employee("Bob", 30);

        emp1.displayInfo();
        emp2.displayInfo();
        emp3.displayInfo();
    }
}

// Output:
// Name: Unknown, Age: 0
// Name: Alice, Age: 0
// Name: Bob, Age: 30

Example 2: Constructor Chaining with Parent Class


class Person {
    private String name;

    // Constructor in parent class
    public Person(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }
}

class Student extends Person {
    private int grade;

    // Constructor in child class
    public Student(String name, int grade) {
        super(name); // Calls the constructor of the parent class
        this.grade = grade;
    }

    public void displayInfo() {
        System.out.println("Name: " + getName() + ", Grade: " + grade);
    }

    public static void main(String[] args) {
        Student student = new Student("Charlie", 10);
        student.displayInfo();
    }
}

// Output:
// Name: Charlie, Grade: 10
