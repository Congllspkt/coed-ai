Loops in Java

What are Loops?

Loops in Java are used to execute a block of code repeatedly as long as a specified condition is true. Java provides several types of loops, including for, while, and do-while.

Example Usage

For Loop

for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}

While Loop

int i = 0;
while (i < 5) {
    System.out.println("Iteration: " + i);
    i++;
}

Do-While Loop

int i = 0;
do {
    System.out.println("Iteration: " + i);
    i++;
} while (i < 5);

How it Works

Each type of loop has its own use case:

  • For Loop: Best for iterating a fixed number of times.
  • While Loop: Best for iterating as long as a condition is true.
  • Do-While Loop: Similar to while, but guarantees at least one execution of the loop body.
