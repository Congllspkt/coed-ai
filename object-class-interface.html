Object, Class, Interface, and @Interface in Java

Object

In Java, an object is an instance of a class. It contains state (fields) and behavior (methods).

Class

A class is a blueprint for creating objects. It defines fields and methods that the objects of the class will have.

Interface

An interface in Java is a reference type that contains abstract methods. It is used to achieve abstraction and multiple inheritance.

@Interface

@Interface is used to define custom annotations in Java. It is a special kind of interface.

Example

// Example of Object, Class, Interface, and @Interface
interface MyInterface {
    void display();
}

class MyClass implements MyInterface {
    private String name;

    MyClass(String name) {
        this.name = name;
    }

    @Override
    public void display() {
        System.out.println("Name: " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass("Example");
        obj.display();
    }
}
