Continue in Java

What is Continue?

The continue statement in Java is used to skip the current iteration of a loop and proceed to the next iteration. It is commonly used in for, while, and do-while loops.

Example Usage

for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue; // Skip the iteration when i is 2
    }
    System.out.println("Iteration: " + i);
}

In the above example, the continue statement skips the iteration when i is equal to 2, and the loop proceeds with the next iteration.

How it Works

When the continue statement is encountered, the control jumps to the next iteration of the loop, bypassing the remaining code in the current iteration.
