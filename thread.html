Thread in Java

What is a Thread?

A thread in Java is a lightweight process that allows a program to perform multiple tasks simultaneously. Threads are part of the Java concurrency framework.

Example of Thread in Java

// Example of Thread in Java
class MyThread extends Thread {
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println(Thread.currentThread().getName() + " - " + i);
        }
    }
}

public class ThreadExample {
    public static void main(String[] args) {
        MyThread thread1 = new MyThread();
        MyThread thread2 = new MyThread();

        thread1.start();
        thread2.start();
    }
}
