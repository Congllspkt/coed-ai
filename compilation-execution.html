Compilation & Execution in Java

What is Compilation & Execution?

In Java, the process of running a program involves two main steps: compilation and execution. Compilation converts the source code into bytecode, while execution runs the bytecode on the Java Virtual Machine (JVM).

Steps for Compilation & Execution

1. Compilation

Use the javac command to compile a Java file:

javac HelloWorld.java

This generates a HelloWorld.class file containing the bytecode.

2. Execution

Use the java command to execute the compiled bytecode:

java HelloWorld

This runs the program on the JVM.

Example Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Save the above code in a file named HelloWorld.java, then compile and execute it using the steps mentioned above.
