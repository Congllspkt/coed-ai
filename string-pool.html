String Pool in Java

What is the String Pool?

The String Pool is a special memory region in Java where string literals are stored. It helps in saving memory by reusing immutable string objects.

Example of String Pool in Java

// Example of String Pool
public class StringPoolExample {
    public static void main(String[] args) {
        String str1 = "Hello";
        String str2 = "Hello";
        String str3 = new String("Hello");

        System.out.println(str1 == str2); // true, both refer to the same object in the pool
        System.out.println(str1 == str3); // false, str3 refers to a new object
    }
}
