Break in Java

What is Break?

The break statement in Java is used to terminate a loop or switch statement prematurely. When encountered, it immediately exits the loop or switch block.

Example Usage

for (int i = 0; i < 5; i++) {
    if (i == 3) {
        break; // Exit the loop when i is 3
    }
    System.out.println("Iteration: " + i);
}

In the above example, the break statement exits the loop when i is equal to 3, and the remaining iterations are skipped.

How it Works

When the break statement is executed, the control jumps out of the loop or switch block, bypassing any remaining code in the block.
