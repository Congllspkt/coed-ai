@RequestHeader Annotation in Spring Boot

What is @RequestHeader?

The @RequestHeader annotation in Spring Boot is used to bind HTTP request header values to method parameters in a controller. It allows you to access specific header values in your application.

Example Usage

@RestController
public class UserController {

    @GetMapping("/users")
    public ResponseEntity<String> getUserAgent(@RequestHeader("User-Agent") String userAgent) {
        // Process the User-Agent header
        return ResponseEntity.ok("User-Agent: " + userAgent);
    }
}

In the above example, the @RequestHeader annotation is used to bind the User-Agent header value to the userAgent method parameter.

How it Works

When a request is made to the endpoint, Spring automatically maps the specified header value to the method parameter annotated with @RequestHeader. If the header is not present, a default value can be specified or an exception will be thrown if the header is required.
