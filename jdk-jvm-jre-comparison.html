Comparison of JDK, JVM, and JRE

JDK (Java Development Kit)

The JDK is a software development kit used to develop Java applications. It includes the JRE and development tools like the compiler (javac).

// Example of compiling a Java program
javac HelloWorld.java
java HelloWorld

JVM (Java Virtual Machine)

The JVM is a runtime environment that executes Java bytecode. It provides platform independence by translating bytecode into machine code.

// Example of JVM execution
java HelloWorld

JRE (Java Runtime Environment)

The JRE provides libraries and the JVM to run Java applications. It does not include development tools like the compiler.

// Example of running a Java program
java -jar MyApp.jar
