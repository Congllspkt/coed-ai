Memory in Java

Memory Management in Java

Java uses an automatic memory management system called Garbage Collection. The memory is divided into different regions such as Heap, Stack, and Method Area.

Example of Memory Usage in Java

// Example of Memory Usage
public class MemoryExample {
    public static void main(String[] args) {
        int localVariable = 10; // Stored in Stack
        Example obj = new Example(20); // Object stored in Heap
    }
}

class Example {
    int value;

    Example(int value) {
        this.value = value;
    }
}
