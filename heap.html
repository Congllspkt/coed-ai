Heap in Java

What is a Heap?

The heap is a region of memory used for dynamic memory allocation. Objects in Java are always allocated on the heap, and the garbage collector manages the heap memory.

Example of Heap Usage in Java

// Example of Heap memory usage
class Example {
    int value;

    Example(int value) {
        this.value = value;
    }

    public static void main(String[] args) {
        Example obj = new Example(10); // Object allocated on the heap
        System.out.println("Value: " + obj.value);
    }
}
