Delegate in Java

What is Delegation?

Delegation in Java is a design principle where an object hands over the responsibility of executing a task to another object. It is commonly used to achieve code reuse and separation of concerns.

Example Usage

class Printer {
    public void print(String message) {
        System.out.println(message);
    }
}

class Manager {
    private Printer printer = new Printer();

    public void delegatePrint(String message) {
        printer.print(message); // Delegating the task to Printer
    }
}

public class DelegationExample {
    public static void main(String[] args) {
        Manager manager = new Manager();
        manager.delegatePrint("Hello, Delegation!");
    }
}

In the above example, the Manager class delegates the task of printing a message to the Printer class.

How it Works

Delegation works by creating an instance of the helper class (e.g., Printer) inside the delegating class (e.g., Manager) and calling the helper class's methods to perform specific tasks.
