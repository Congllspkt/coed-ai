Stack in Java

What is a Stack?

A stack is a data structure that follows the Last In, First Out (LIFO) principle. It is commonly used in programming for tasks like expression evaluation, backtracking, and more.

Example of Stack in Java

// Example of using Stack in Java
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack stack = new Stack<>();
        stack.push(10);
        stack.push(20);
        stack.push(30);

        System.out.println("Top element: " + stack.peek());
        System.out.println("Popped element: " + stack.pop());
        System.out.println("Stack after pop: " + stack);
    }
}
