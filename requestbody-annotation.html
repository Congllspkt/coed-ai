@RequestBody Annotation in Spring Boot

What is @RequestBody?

The @RequestBody annotation in Spring Boot is used to bind the HTTP request body to a Java object. It is commonly used in RESTful web services to handle incoming JSON or XML payloads.

Example Usage

@RestController
public class UserController {

    @PostMapping("/users")
    public ResponseEntity<String> createUser(@RequestBody User user) {
        // Process the user object
        return ResponseEntity.ok("User created: " + user.getName());
    }
}

In the above example, the @RequestBody annotation is used to map the incoming JSON payload to the User object.

How it Works

When a request is made to the endpoint, Spring automatically deserializes the request body into the specified Java object using the configured message converters (e.g., Jackson for JSON).
