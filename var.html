var in Java

What is var?

The var keyword in Java, introduced in Java 10, allows you to declare local variables without explicitly specifying their type. The type is inferred by the compiler based on the value assigned to the variable.

Example Usage

public class VarExample {
    public static void main(String[] args) {
        var message = "Hello, var!"; // Compiler infers type as String
        var number = 42; // Compiler infers type as int

        System.out.println(message);
        System.out.println("Number: " + number);
    }
}

In the above example, the var keyword is used to declare variables without explicitly specifying their types.

How it Works

The var keyword can only be used for local variables with an initializer. The compiler determines the type of the variable at compile time based on the assigned value.
