Generics in Java

What are Generics?

Generics in Java allow you to define classes, interfaces, and methods with type parameters. They provide type safety by enabling you to specify the type of objects that a class, interface, or method can work with.

Example Usage

Generic Class

public class Box {
    private T item;

    public void setItem(T item) {
        this.item = item;
    }

    public T getItem() {
        return item;
    }
}

In the above example, the Box class is a generic class that can hold any type of object.

Using the Generic Class

Box stringBox = new Box<>();
stringBox.setItem("Hello Generics");
System.out.println(stringBox.getItem());

How it Works

Generics are implemented using type erasure, which means that the type parameters are replaced with their bounds or Object during compilation. This ensures backward compatibility with older versions of Java.
