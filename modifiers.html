Access Modifiers in Java

Access Modifiers

Access modifiers in Java determine the scope and visibility of a class, method, or variable. The main access modifiers are:

  • private: Accessible only within the same class.
  • default: Accessible within the same package.
  • protected: Accessible within the same package and subclasses.
  • public: Accessible from anywhere.

Other Modifiers

  • static: Belongs to the class rather than an instance.
  • final: Prevents modification of a variable, method, or class.

Example of Modifiers in Java

// Example of Modifiers in Java
public class ModifiersExample {
    private int privateVar = 10;
    protected int protectedVar = 20;
    public int publicVar = 30;
    static int staticVar = 40;
    final int finalVar = 50;

    public static void main(String[] args) {
        ModifiersExample obj = new ModifiersExample();
        System.out.println("Private: " + obj.privateVar);
        System.out.println("Protected: " + obj.protectedVar);
        System.out.println("Public: " + obj.publicVar);
        System.out.println("Static: " + staticVar);
        System.out.println("Final: " + obj.finalVar);
    }
}
